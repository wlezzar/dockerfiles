# JMXTrans for Kafka
A JMX Trans image ready to be plugged to a kafka instance 0.8.2 or later

# Configuration
In addition to the base envrionment variables, you may use these environment variables to customise the jmxtrans run :
- KAFKA_JMX_PORT
- KAFKA_JMX_HOST
- INFLUXDB_HOST
- INFLUXDB_PORT
- INFLUXDB_DBNAME
- INFLUXDB_USER
- INFLUXDB_PASSWORD
